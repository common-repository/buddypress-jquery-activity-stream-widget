== Description ==

<h1>Widgetize your BP activity streams! </h1>
<br />
Let your site viewers/users easily read the activity streams by adding a simple yet customizable widget that displays streams in an animated manner.  
<br />
<b>Features</b>:
<ul>
<li>Full animation speed control</li>
<li>Full animation timeout speed control</li>
<li>Select from two effects (Slide and Fade). Will be adding more soon!</li>
<li>Customizable widget area.</li>
</ul>
<font color="red">
Note: This plugin works with <a href="http://buddypress.org/">buddypress</a> installed on your website. 
</font>

== Installation ==


1. Upload this plugin to `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In your wordpress administration, go to appearance>widgets>. Drag and drop the plugin in widgets area.
<br />
<br />
<br />
<font color="red">
Note: Your theme must be widget ready.
</font>
